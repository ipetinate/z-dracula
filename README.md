# Z Dracula Theme for Zed

Theme extension for the Zed editor.

## Structure

- `extension.toml`
- `themes/z-dracula.json`

## How to use locally (Dev Extension)

1. Open Zed.
2. Go to `Extensions`.
3. Click `Install Dev Extension`.
4. Select the `z-dracula` folder.
