## Install from Zed Extensions

1. Open the Command Palette with `CMD + SHIFT + P` (macOS) or `CTRL + SHIFT + P` (Windows/Linux), then type "extensions".
2. Run `zed: extensions` from the Command Palette.
3. In the Extensions tab, search for `Z Dracula`.
4. Click `Install` on the `Z Dracula` extension by Isac Petinate.

## Install manually from the repository

### Install with Git

If you use Git, you can install the theme and keep it up to date by cloning the repository:

```bash
git clone https://github.com/ipetinate/z-dracula.git
```

### Install with ZIP

Download the [GitHub `.zip` archive](https://github.com/ipetinate/z-dracula/archive/main.zip) and unzip it.

### Activate the theme

1. Copy `themes/z-dracula.json` to `~/.config/zed/themes`.
2. Select the theme in Zed (`Settings` -> `Select Theme` -> `Z Dracula`).
3. Done.
